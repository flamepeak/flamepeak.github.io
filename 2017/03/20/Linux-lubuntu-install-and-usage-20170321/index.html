Linux lubuntu 安装与使用

By FlamePeak

Published 2017-03-20

lunbuntu 使用lxde界面，速度快，流畅，很少出现卡死情况。

当然安装速度也是飞一般的快

安装

使用

  1. 设置root密码 sudo passwd

  2. 安装chrome浏览器
    在安装chrome是会出现

dpkg: dependency problems prevent configuration of google-chrome-stable:
google-chrome-stable depends on libpango1.0-0 (>= 1.14.0); however:
Package libpango1.0-0 is not installed.
google-chrome-stable depends on fonts-liberation; however:
Package fonts-liberation is not installed.

需要解决依赖关系：
使用dpkg -i soft.deb 的时候出现依赖没有安装
使用apt-get -f -y install 解决依赖问题后再执行dpkg安装deb包

  1. 安装语言包
    preference --> language support

如果想启动系统就自动修复的可以如下：

  1. 安装notify-send
    ubuntu的气泡提示是使用notify-osd来实现的，我们需要下载libnotify
    apt-get install notify-osd (ubuntu 已经自带)
    apt-get install libnotify-dev

  2. 安装7z apt-get install p7zip-full
