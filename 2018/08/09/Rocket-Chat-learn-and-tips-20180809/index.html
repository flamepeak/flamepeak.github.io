Rocket.Chat 学习与使用

By FlamePeak

Published 2018-08-09

开发环境

  1. 安装Git yum install git
  2. 安装Meteor
    curl https://install.meteor.com/ | sh
  3. clone and run

可能遇到的错误

  1. g++ not found

  2. node-pre-gyp ERR! Completion callback never invoked! 这个错误原因未知，花了很长时间解决，最后用了一个简单的办法。 去npmjs网站查看node-pre-gyp的版本

然后，修改Rocket.Chat目录下文件package-lock.json，查找文件目录里面的所有node-pre-gyp，把版本号换成最新的版本
